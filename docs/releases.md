# ニコ生ショートカット

## Release Notes

### v0.4.2 (2026/3/28)

* 以下の不具合を修正しました。
  * `c` のコメント On/Off の表示が実際の状態と逆になる問題
  * `F` のフルスクリーン表示が実際の状態と逆になる問題
  * `P` の `フォロー中の番組` が最新のニコ生視聴ページで動作しない問題
  * `S` の `ユーザーを開く` が最新のニコ生視聴ページで動作しない問題

* 以下のショートカットを削除しました。
  * `C` ... コミュニティを開く

### v0.4.1 (2026/3/26)

* 以下の不具合を修正しました。
  * `m` のミュート On/Off の表示が実際の状態と逆になることがある問題

### v0.4.0 (2022/5/20)

⚠️ ショートカット追加にともない、既存のカスタマイズ設定が初期化されます。手抜きでごめんなさい。⚠️

* 以下のショートカットを追加しました。
  * `p` ... プレイヤー位置までスクロール

### v0.3.0 (2022/5/18)

* 以下のショートカットを追加しました。
  * `U`, `I` (大文字) ... ボリュームを大きく下げる、上げる
  * `J`, `L` (大文字) ... 60秒、戻る、進む

* `ページ読み込み時に強制的にミュート解除 + ボリュームを100%にする` 機能を追加しました。
  * デフォルトでオフなので、設定から手動でオンにしてください。

### v0.2.1 (2022/5/14)

* 初版リリースしました。
